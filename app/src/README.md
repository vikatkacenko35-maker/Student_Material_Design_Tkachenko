# Лабораторная работа №10-11. Создание приложения «Список студентов» с использованием Material Design
 
--- 

# Цель работы:
Научиться применять принципы Material Design в Androidприложениях с использованием Jetpack Compose. В результате лабораторной работы
будет создано приложение «Список студентов», которое демонстрирует использование
цветовых схем, типографики, форм и анимаций Material Design.

# Функциональность
Приложение предоставляет следующий набор возможностей:

- Отображение списка студентов: Главный экран приложения содержит прокручиваемый список из 10 студентов.

- Информация о студенте: Каждый элемент списка (карточка) отображает фотографию, имя, возраст и краткое юмористическое описание.

- Адаптивная цветовая тема: Приложение поддерживает как светлую, так и тёмную тему. Цвета интерфейса автоматически подстраиваются под системные настройки устройства.

- Кастомизированные формы: Фотографии студентов имеют круглую форму, а карточки студентов — уникальную форму со скруглёнными углами (левый нижний и правый верхний).

- Интерактивные карточки: Нажатие на иконку-стрелку в правой части карточки плавно раскрывает или сворачивает дополнительную информацию о студенте с анимацией пружины (spring animation).

- Верхняя панель приложения (TopAppBar): Содержит логотип и название приложения, выполненное кастомным декоративным шрифтом.

# Технологии
В проекте использовались следующие технологии и библиотеки:

- Язык программирования: Kotlin

- Дизайн-система: Material Design 3 (Material You)

- Графика: Material Icons Extended

- Шрифты: Montserrat, Abril Fatface (из Google Fonts)

- Система сборки: Gradle (Kotlin DSL)

- Система контроля версий: Git
